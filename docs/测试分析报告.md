# 测试分析报告

## 测试环境

- Python 3.10+
- matplotlib
- scikit-learn

## 测试命令

```bash
python -m unittest discover tests
```

## 测试内容

| 测试文件 | 测试重点 |
|---|---|
| `test_reader.py` | CSV 读取、字段校验、非法标签处理 |
| `test_cleaner.py` | 文本清洗和关键词提取 |
| `test_sentiment.py` | 正面、负面、中性和否定词处理 |
| `test_problem.py` | 多问题归因和正面评论处理 |
| `test_statistics.py` | 品牌统计、情感分布和问题排行 |
| `test_ml_classifier.py` | sklearn 训练、预测和评估 |
| `test_chart_generator.py` | 图表文件生成 |
| `test_report.py` | CSV 和 Markdown 报告导出 |

## 预期结果

所有测试通过，且主程序能生成：

- `analyzed_reviews.csv`
- `analysis_report.md`
- `ml_evaluation.txt`
- 3 张 PNG 图表
