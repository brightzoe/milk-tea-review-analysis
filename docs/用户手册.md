# 用户手册

## 环境准备

```bash
pip install -r requirements.txt
```

## 数据格式

输入文件默认为 `data/reviews.csv`，字段如下：

```csv
review_id,date,brand,product,channel,content,rating,label
```

`label` 只能填写：`正面`、`中性`、`负面`。

当前样例数据共 500 条，由 ASAP 公开中文餐馆评论 sample 和人工整理的奶茶场景样例混合生成。数据来源说明见 `docs/数据来源说明.md`。

其中，ASAP 公开数据中筛选出的饮品相关评论片段约 32 条，人工整理的奶茶消费场景样例约 468 条。当前数据主要用于课程项目演示，不直接代表真实市场统计结论。

如需重新生成样例数据：

```bash
python scripts/build_dataset.py
```

## 运行系统

```bash
python -m src.main
```

或直接运行入口文件：

```bash
python src/main.py
```

指定输入和输出：

```bash
python -m src.main --input data/reviews.csv --output output
```

## 查看结果

运行后查看：

- `output/analysis_report.md`：分析报告。
- `output/analyzed_reviews.csv`：逐条评论分析结果。
- `output/ml_evaluation.txt`：机器学习模型评估。
- `output/sentiment_model.pkl`：保存的 sklearn 情感分类模型。
- `output/charts/`：图表。

## 运行测试

```bash
python -m unittest discover tests
```

测试通过后，说明 CSV 读取、文本清洗、情感分类、问题归因、统计、图表和报告导出等主要功能可正常运行。

当前版本测试结果为：

```text
Ran 22 tests
OK
```

## 常见错误

- CSV 缺字段：检查表头是否完整。
- label 非法：只能使用正面、中性、负面。
- 图表中文显示异常：不影响分析结果，可在本地安装中文字体后重新运行。
